# Nebula Browser — Test Document

This is a test file used to verify the drag-and-drop file-attach feature.

## Key capabilities

- Drag files from the OS onto the browser window
- Files open in a new tab with appropriate renderer (image / video / pdf / text)
- Files dropped onto the AI sidebar are attached to the conversation
- Downloads panel tracks every file you've opened this session
